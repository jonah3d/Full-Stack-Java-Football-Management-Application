Gestión Deportiva

Esta aplicación fue creada como un proyecto individual para mi último año de DAM.

El propósito es desarrollar una aplicación que pueda gestionar una lista de equipos y sus jugadores, y hacer que los datos sean persistentes.

Esta aplicación fue creada completamente en Java con Swing con la ayuda de la base de datos Oracle para mantener los datos.
